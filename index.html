ETIQUETAS HTML


1- Etiqueta <a>

define un hipervinculo/link/enlace..etc

Atributos:download, href, hreflang, media, ping, referrerpolicy, rel, target, type

Ejemplo

sigueme en <a href="https://ezio2220.github.io/TPI/"> Github</a>

2- Etiqueta <abbr>

define una abreviacion o un acronimo

Atributos:title.

Ejemplo

EL <abbr title="World Wide Web Consortium">W3C</abbr> fue fundado en 1994.

Resultado

EL W3C fue fundado en 1994.

3- Etiqueta <address>

define la informacion de contacto del autor o dueño de un documento

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Resultado

sigueme en:
github.com/Ezio2220

4- Etiqueta <area>

define un area dentro de un mapa de imagen

Atributos:alt, coords, download, href, hreflang, media, rel, shape, target, type

Ejemplo

<img src="planetas.gif" width="145" height="126" alt="Planetas" usemap="#planetmap"> <map name="planetas"> <area shape="rect" coords="0,0,82,126" download="sol.jpg" > <area shape="circle" coords="90,58,3" download="mercurio.jpg" > </map>

Resultado

5- Etiqueta <b>

define texto en negrita

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

Resultado

6- Etiqueta <base>

especifica la url bas/ destino para todas las urls relativas de un documento

Atributos: href, target

Ejemplo

Resultado

7- Etiqueta <bdo>

anula la direccion del texto actual

Atributos:dir

Ejemplo:

<bdo dir="rtl"> este texto ira de derecha a izquierda <bdo>

Resultado

este texto ira de derecha a izquierda

8- Etiqueta <blockquote >

defune una secion que se cita de otra fuente

Atributos:cite

Ejemplo

<blockquote cite="github.com/Ezio2220"> este trabajo es demasiado largo y da sueño seguir... hasta hambre </blockquote>

Resultado

este trabajo es demasiado largo y da sueño seguir... hasta hambre

9- Etiqueta <button>

define un boton clckeable

Atributos:autofocus, disabled, form, formaction, formenctype, formnovalidate, formtarget, name type value /p>

Ejemplo

<button type="button">Clickeame </button>

Resultado

ETIQUETAS HTML 5




1- Etiqueta <audio>

define un contenido sonoro.

Atributos:autoplay, controls, loop, muted, preload, src

Ejemplo

<audio controls> <source src="hola.mp3" type="audio/mpeg"> </audio>

Resultado

2- Etiqueta <article>

define un hipervinculo/link/enlace..etc

Atributos:download, href, hreflang, media, ping, referrerpolicy, rel, target, type

Ejemplo

sigueme en <a href="https://github.com/Ezio2220"> Github</a>

Resultado

sigueme en Github;

3- Etiqueta <aside>

define un contenido a lado del contenido de la pagina

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<p> contenido </p> <aside> <h5> otra onda</h5> <p>si otra onda</p> </aside>

Resultado

contenido

otra onda

si otra onda

4- Etiqueta <canvas>

se usa para hacer dibujos, sobre la marcha mediante scripts (usualmente javascript)

Atributos:height, width

Ejemplo

<canvas id="Canvas1"></canvas> <script> var canvas = document.getElementById("Canvas1"); var ctx = canvas.getContext("2d"); ctx.fillStyle = "#FF0000";ctx.fillRect(0, 0, 80, 80); </script>

Resultado

5- Etiqueta <details>

define detalles adicionales que el usuario puede ver o ocultar.

Atributos:open.

Ejemplo

<summary>Leonardo da Vinci.</summary> <details> <p>en la actualidad no existe!. </p> </details>

Resultado

Leonardo da Vinci.

en la actualidad no existe!.

6- Etiqueta <datalist>

especifica una lista de opciones predefinidas para los input.

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<input list="browsers"> <datalist id="browsers"> <option value="Internet Explorer"> <option value="Firefox"> <option value="Chrome"> <option value="Opera"> <option value="Safari"> </datalist>

Resultado

7- Etiqueta <embed>

define un contenedor para una aplicacion(no html) externa.

Atributos:height, src, type, width

Ejemplo

<embed src="holamundo.swf">

Resultado

8- Etiqueta <foter>

define el pie de pagina para un documento o seccion

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<footer> <p>hecho por: Franklin Fuentes</p> </footer>

Resultado

hecho por: Franklin Fuentes

9- Etiqueta <header>

define un encabezado para un documento o seccion

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<header> <h5>un encabezado importante</h5> </header>

Resultado

un encabezado importante

10- Etiqueta <mark>

define un texto marcado/ resaltado

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<p>estudia para <mark>ingenieria economica</mark> Hoy.</p>

Resultado

estudia para ingenieria economica Hoy.

11- Etiqueta <meter>

define una medidor dentro de un rango definido.

Atributos:form, high, low, max, min, optimum, value.

Ejemplo

<meter value="0.6">60%</meter>

Resultado

60%

12- Etiqueta <nav>

define los links de navegacion del sitio.

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<nav> <a href="#HTML">Etiquetas Html5</a> </nav>

Resultado

Etiquetas Html5

13- Etiqueta <progress>

representa el progreso de una tarea(barrita de carga).

Atributos:max, value

Ejemplo

<progress value="22" max="100"></progress>

Resultado

14- Etiqueta <svg>

define un contenedor de graficos svg(graficas vectoriales escalables)

Atributos:width, height

Ejemplo

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="green" stroke-width="4" fill="yellow" /> </svg>

Resultado

15- Etiqueta <source>

define multiples recursos multimedia para elementos de multimedia(audio y video)

Atributos: src, srcset, media, sizes, type.

Ejemplo

<audio controls> <source src="hola.mp3" type="audio/mpeg"> </audio>

Resultado

16- Etiqueta <summary>

define un encabezado visible para los elementos de la etiqueta <details>

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

Ejemplo

<summary>Leonardo da Vinci.</summary> <details> <p>en la actualidad no existe!. </p> </details>

Resultado

Leonardo da Vinci.

en la actualidad no existe!.

17- Etiqueta <track>

define pistas de texto para elementos multimedia.

Atributos:default, kind, label, src, srclang

Ejemplo

<video width="160" height="120" controls> <source src="trailer.mp4" type="video/mp4"> <track src="subtitles_no.vtt" kind="subtitles" srclang="no" label="English"> </video>

Resultado

18- Etiqueta <time>

define una fecha o hora.

Atributos:datetime

Ejemplo

<p>En esta fecha <time datetime="2020-01-10 10:00">Viajaré</time>.</p>

Resultado

En esta fecha Viajaré.

19- Etiqueta <video>

define un video o pelicula.

Atributos:autoplay, controls, height, loop, muted, poster, preload, src, width

Ejemplo

<video width="160" height="120" controls> <source src="trailer.mp4" type="video/mp4"> <track src="subtitles_no.vtt" kind="subtitles" srclang="no" label="English"> </video>

Resultado

define un posible salto de linea

Atributos:este elemento solo posee los atributos globales y de eventos.(Vease al inicio)

<p> para ser mejor debes dejar de compararte con<wbr>otros<wbr>centrate solo en ti mismo. </p>
